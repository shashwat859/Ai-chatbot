# AI Chatbot Model

This repository contains the implementation of an AI Chatbot Model.

## Project Structure
- `requirements.txt`: Contains the required packages.
- `model.py`: Main model training file.

## Getting Started

### Prerequisites
- Python 3.x
- See `requirements.txt` for the list of packages.

### Installation
1. Clone the repo.
2. Install dependencies using `pip install -r requirements.txt`.

### Usage
Run the training script using `python model.py`.
